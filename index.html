HTML Syntax

HTML doesn't take too long to grasp. The basics are simple. Opening brackets consist of an arrow (<), a tag like 'body', 'p' for paragraph or the different headings like 'h1', 'h2', 'h3', etc which become smaller as the number goes up, and another arrow (>).

You can't have opening tags without your closing tags though. Closing tags are similar to opening tags, but with one important distinction. Before the tag you put a forward slash (/).

CSS Syntax

This website is quite beautiful, and most websites are made to look appealing because otherwise people wouldn't want to look at them.

The basic HTML for websites though isn't as appealing, with basic fonts and colors. You can doewnload other fonts and colors to put into your website but sometimes it just doesn't cut it.

That's where CSS, or Cascading Style Sheets, comes into play.

Linking your CSS to HTML files

To be able to use a CSS file with your HTML file, you must first link the CSS file to the HTML file using the <link> tag in HTML. Within this you include the relation using rel="stylesheet", the type with type="text/css", and finally the part that actually links it, the href> with the file path.

Once you've linked your files you can start coding in CSS. There are many properties to play with like font-family, font-size, font-weight, color, background-color, etc. All of these change how your elements in HTML appear in your browser

Some code in CSS

For CSS, you put the identifier like html, body, h1, etc and then a set of curly brackets. Within those curly brackets you can put whatever style rules you like. An important one is color and making sure you pick colors that work well together, like I have tried to do here.
